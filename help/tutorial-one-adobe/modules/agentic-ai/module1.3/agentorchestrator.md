# 1.1 Agent Orchestrator

## Lernziele

Erfahren Sie, wie Sie Adobe Experience Platform-Agenten und Agent Orchestrator für Folgendes verwenden:

- Kauftrends analysieren
- Identifizieren von Zielgruppen mit hoher Neigung
- Validieren der Journey-Leistung
- Erstellen Sie eine neue Journey für den CitiSignal Fiber Max Rollout

## Voraussetzungen

- Zugriff auf Adobe Journey Optimizer und Customer Journey Analytics
- Vertrautheit mit [Agent Orchestrator-Konzepten](https://experienceleague.adobe.com/de/docs/experience-cloud-ai/experience-cloud-ai/agents/agent-orchestrator)
- Zugriff auf CitiSignal-Demodaten: Bestellungen, Bestellungen, Zielgruppen, Journey
