# 1.7 Benutzerdefinierte Firefly-Workflows

Der Schwerpunkt von Firefly Custom Workflows liegt auf kreativen Technologen und Lösungsentwicklern, die Workflows erstellen und konfigurieren.

Benutzerdefinierte Firefly-Workflows ermöglichen Folgendes:

- Erstellen von Workflows: Erstellen Sie mehrstufige Produktions-Workflows mithilfe einer visuellen Oberfläche mit manueller oder KI-Unterstützung.
- Workflows ausführen: Führen Sie Workflows im benötigten Umfang aus, einschließlich Batch-Verarbeitung, mit einer robusten Laufzeit, die außerhalb des Browsers funktioniert.
- Workflows bereitstellen: Veröffentlichen Sie Workflows als API-Endpunkte oder programmgesteuerte Benutzeroberflächen zur Integration in andere Tools und Systeme.
- Workflows integrieren: Verbinden von Workflows mit Unternehmenssystemen wie DAMs, Aktivierungsdatenströmen und Inhalts-Lieferketten.

Die unterstützten primären Anwendungsfälle umfassen Dinge wie:

- Anpassungen von Social-Media-Kanälen, z. B. das Erstellen von Größenvarianten für jede Social-Media-Plattform und jeden Typ.
- Asset-Vorbereitung, z. B. Vorbereiten von Assets für die Verwendung in zusätzlichen Oberflächen wie Drucken oder Web.
- Content-Retargeting, z. B. Aktualisieren und Wiederverwenden von Visualisierungen, um erneut mit interessierten Zielgruppen zu interagieren.
- E-Commerce, z. B. die Normalisierung von Produktbildern und die Erstellung segmentspezifischer Varianten für verschiedene Zielgruppen.
