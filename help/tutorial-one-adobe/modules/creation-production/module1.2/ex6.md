# 1.2.6 Frame I/O zu Workfront Fusion zu AEM Assets

In der vorherigen Übung haben Sie ein Szenario konfiguriert, das mithilfe von Adobe Firefly automatisch Varianten einer Adobe Photoshop PSD-Datei generiert. Die Ausgabe dieses Szenarios war eine neue Photoshop PSD-Datei. Die Business-Teams benötigen jedoch keine PSD-Datei, sondern eine PNG-Datei oder eine JPG-Datei. In dieser Übung konfigurieren Sie eine neue Automatisierung, die dazu führt, dass eine PNG-Datei generiert wird, sobald das Asset in Frame I/O genehmigt wurde, und dass diese PNG-Datei automatisch in AEM Assets gespeichert wird.
