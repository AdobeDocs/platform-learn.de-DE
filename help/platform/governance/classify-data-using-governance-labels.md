# Daten mithilfe von Governance-Beschriftungen klassifizieren

Erfahren Sie, wie Sie Governance-Beschriftungen auf Ihre Datensätze anwenden. Anhand von Governance-Beschriftungen können Sie Daten taggen und klassifizieren, um Unternehmensrichtlinien, vertragliche Verpflichtungen, Compliance-Anforderungen und regionale Vorschriften in Adobe Experience Platform widerzuspiegeln. Diese Fähigkeit ist von entscheidender Bedeutung, um bekannte und unbekannte Daten zu Ihren Kunden zu unterscheiden und die Datennutzung entsprechend der Art der Daten angemessen zu steuern. Eine ausführliche Produktdokumentation finden Sie unter [Verwalten von Datennutzungsbezeichnungen in der Benutzeroberfläche](https://experienceleague.adobe.com/docs/experience-platform/data-governance/labels/user-guide.html?lang=de) und [Verwalten von Datennutzungsbezeichnungen mit API](https://experienceleague.adobe.com/docs/experience-platform/data-governance/labels/dataset-api.html)
